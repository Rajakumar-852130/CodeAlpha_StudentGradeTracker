# Student Grade Tracker - CodeAlpha Internship

A professional, modern web application built during my Java Programming Internship at **CodeAlpha**. This project helps teachers and students manage academic records by tracking grades and automatically calculating performance statistics.

## 🚀 Features
- **Add Students**: Easily input student names and their respective grades.
- **Real-time Statistics**: Automatically calculates:
    - **Average Score** of the class.
    - **Highest Score** achieved.
    - **Lowest Score** recorded.
- **Premium UI**: Dark-themed, responsive dashboard with glassmorphism effects and smooth animations.
- **Pass/Fail Indicators**: Visual badges to identify student status at a glance.
- **Memory Efficient**: Uses Java's `ArrayList` for high-performance in-memory data management.

## 🛠️ Technology Stack
- **Backend**: Java 17, Spring Boot 3.x
- **Frontend**: HTML5, CSS3 (Vanilla), Thymeleaf
- **Build Tool**: Maven

## 💻 How to Run
1. Ensure you have **Java 17 or higher** installed.
2. Clone this repository or download the ZIP.
3. Navigate to the project folder.
4. Double-click on `run_project.bat` (on Windows) or run the command:
   ```bash
   ./mvnw spring-boot:run
   ```
5. Open your browser and go to: `http://localhost:8081`
